Onde podemos escrever códigos css:
Externa - separadas do arquivo html == .css - todos os documentos
Interna - juntos do arquivo html => <head> </head> - somente a página em questão
Inline - juntos do arquivo html é na mesma linha == <h1>Hello</h1> - somente a linha de código
